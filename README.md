📋 Phần mềm Quản lý Nhân sự cơ bản (WinForms & SQL LocalDB)
Đây là một ứng dụng Desktop cơ bản được xây dựng bằng ngôn ngữ C# trên nền tảng Windows Forms. Ứng dụng cho phép quản lý danh sách nhân viên với đầy đủ các tính năng cơ bản (CRUD).

✨ Tính năng chính
• Hiển thị danh sách: Tự động nạp dữ liệu nhân viên từ Database khi khởi động.

• Thêm nhân viên: Thêm mới nhân viên kèm theo Họ tên và Số điện thoại.

• Sửa thông tin: Click chọn nhân viên trên bảng để cập nhật lại thông tin nhanh chóng.

• Xoá nhân viên: Xoá bỏ nhân viên khỏi hệ thống với thông báo xác nhận an toàn.

• Tự động hoá: Tự động tạo bảng dữ liệu nếu chưa tồn tại trong SQL Server.

🛠 Công nghệ sử dụng
• Ngôn ngữ: C# (.NET Framework / .NET Core)

• Giao diện: WinForms (Windows Forms App)

• Cơ sở dữ liệu: SQL Server LocalDB (sử dụng database hệ thống master để tối ưu quyền truy cập).

• Thư viện: Microsoft.Data.SqlClient hoặc System.Data.SqlClient.

🚀 Hướng dẫn cài đặt & Chạy
• Clone dự án: git clone https://github.com/username/QuanLyNhanSu.git

• Mở bằng Visual Studio: Mở file .sln.

• Cấu hình Database: Ứng dụng sử dụng (LocalDB)\MSSQLLocalDB. Đảm bảo bạn đã cài đặt thành phần này trong Visual Studio Installer.

• Nhấn F5: App sẽ tự động tạo bảng và nạp dữ liệu mẫu để bạn trải nghiệm ngay.
